## Contents

- **Ashtanga Yoga** — Full Primary Series standing sequence with Sanskrit names, breath cues, alignment notes, and pose images. Includes Surya Namaskar A & B (5 rounds each) and all 16 standing asanas.
- **Hatha Yoga** — 13-step teacher class sequence guide, Hatha Surya Namaskar (24 positions with mantras), and 11 standing asanas drawn from the Nirvana Yoga School manual.
- **Practicums** — Guided teaching practicum references for Hatha Yoga, Alignment & Adjustment, and Anapana Sati meditation.

## Tech

Plain HTML + CSS — no build tools, no dependencies, no frameworks. All fonts loaded from Google Fonts. All images base64-encoded inline.

## Deploying to GitHub Pages

1. Push this repo to GitHub
2. Go to your repo → **Settings** → **Pages**
3. Under **Build and deployment**, set:
   - **Source:** `Deploy from a branch`
   - **Branch:** `main` (or `master`) → `/ (root)`
4. Click **Save**
5. Your site will be live at `https://<your-username>.github.io/<repo-name>/`

Any future `git push` to the main branch will automatically redeploy.

## Local Preview

Just open `index.html` in any browser — no server needed.
